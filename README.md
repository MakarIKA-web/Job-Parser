# 🟢 Job Parser & Summarizer Project

This project collects job postings from Arbeidsplassen NAV, summarizes them with Google Gemini AI, and provides an interactive CLI browser for filtering and viewing jobs.

It consists of three main scripts:

- `scraper.py` — Collects raw job listings.  
- `summarizer.py` — Summarizes jobs using AI.  
- `job_browser.py` — CLI tool to filter, sort, and explore jobs.  

## 📂 Project Structure  
project/  
│  
├─ scraper.py          # Collects job links and scrapes details  
├─ summarizer.py       # Summarizes job postings using Google Gemini AI  
├─ job_browser.py      # CLI interface for filtering and viewing jobs  
├─ jobs.json           # Raw scraped jobs (generated by scraper.py)  
└─ result.json         # Summarized jobs (generated by summarizer.py)  

## 🟢 File Descriptions

### 1️⃣ scraper.py
**Purpose:**  
Collects job postings from Arbeidsplassen NAV and saves them in a structured JSON format.

**Key Features:**  
- Uses Playwright to collect job links from dynamically-loaded pages.  
- Uses BeautifulSoup to scrape job details: title, company, location, employment type, work time, language, number of positions, deadline, description, and apply link.  
- Stores all jobs in `jobs.json`.  

**Output:** `jobs.json` — a list of job dictionaries.

---

### 2️⃣ summarizer.py
**Purpose:**  
Uses Google Gemini AI to summarize and enrich raw job postings.

**Key Features:**  
- Reads jobs from `jobs.json`.  
- Sends job descriptions to Gemini AI with a prompt to extract:  
  - Short summary (2–3 sentences)  
  - 5 key requirements  
  - 5 key responsibilities  
  - 3 candidate questions  
- Saves the enriched and structured jobs into `result.json`.  

**Output:** `result.json` — summarized and enriched job postings with fields:
```json
{
    "title": "...",
    "company": "...",
    "location": "...",
    "employment_type": "...",
    "summary": "...",
    "key_requirements": [...],
    "key_responsibilities": [...],
    "call_questions": [...],
    "url": "..."
}
```

## 3️⃣ job_browser.py

Purpose:
Interactive command-line tool to filter, sort, and view jobs.

Key Features:

Supports filtering by:

City

Employment type

Job title keywords

Logical operator: AND / OR

Sorts jobs by:

Deadline

Job title

Company

Displays top N results (configurable by user).

Allows users to see application links for selected jobs.

Input: jobs.json or result.json
Output: Filtered and sorted job listings printed in CLI.

⚙️ Requirements

Python 3.10+

Libraries:

pip install playwright beautifulsoup4 requests google-genai

For Playwright:

playwright install

Google Gemini API key

🚀 Usage
Step 1 — Scrape jobs
python scraper.py

Scrapes jobs from Arbeidsplassen NAV.

Saves raw job data in jobs.json.

Step 2 — Summarize jobs
python summarizer.py

Reads jobs.json.

Uses Google Gemini AI to summarize jobs.

Saves enriched jobs in result.json.

Step 3 — Explore jobs
python job_browser.py

Loads jobs.json or result.json.

Prompts for filters, sorting, and top N results.

Displays jobs and allows viewing application links.

⚡ Example Workflow

Run scraper:

Collected 120 links
Saved 120 jobs to jobs.json

Run summarizer:

Summarizing 10 jobs...
Saved 10 summarized jobs to result.json

Run job browser:

Available cities: Oslo, Porsgrunn, Skien, Bamble
Available employment types: Full-time, Part-time
Some job title keywords: Software, Engineer, Manager, Teacher

Apply filters and sorting

View top N jobs

Get application link by number

📌 Notes

Ensure UTF-8 console support if using Windows.

Adjust the number of jobs processed in summarizer.py (job_list[:10]) as needed.
